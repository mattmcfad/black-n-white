# Black n White

## About
Using NodeJS to convert images to Black n White

## Technologies
* NodeJS
* [GraphicsMagic](https://github.com/aheckmann/gm)
* Gulp
* NPM
* Stylus

## Getting Started
1. Install grunt dependencies
	
	```
	npm install
	```
2. Update package.json
	* Update your Name, Description, Repository, Homepage etc.
3. Run Gulp
	
	``` 
	gulp
	```
